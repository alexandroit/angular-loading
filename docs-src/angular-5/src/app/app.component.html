@revivejs/angular-loading

A maintained Angular wrapper around the ReviveJS loading toolkit. Use a component for direct templates, a directive for cards and buttons, and a service for fullscreen or ad-hoc loading states.

Component API Render loaders declaratively or wrap projected content.
Directive API Attach loading behavior to cards, tables, and buttons.
Service API Create fullscreen or programmatic loaders when the template is not enough.
Core fidelity The wrapper passes through the same modern variants and anti-flicker options.
See demos README
